This project has been developed solely for demonstration purposes to showcase cyber attack techniques in an academic setting. It is intended to support educational learning and cybersecurity awareness within the framework of ethical hacking and responsible security practices.

This initiative is conducted on behalf of the WiCyS Surrey Student Chapter, aiming to provide students with a practical understanding of cybersecurity threats and defensive strategies. The project adheres strictly to ethical guidelines and legal boundaries, emphasizing the importance of cybersecurity education and responsible hacking practices.

Disclaimer: This project is for educational use only and should not be used for malicious activities.
